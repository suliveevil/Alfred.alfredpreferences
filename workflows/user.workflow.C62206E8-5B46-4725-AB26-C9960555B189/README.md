
集成Github、掘金、知乎、淘宝等智能匹配搜索的一款 alfred workflow 插件。(不依赖第三方模块)

![](http://oqhtscus0.bkt.clouddn.com/f27594afeda6b513ffec98c3e60ccbb0.jpg-400)


## 使用方法:

* 双击 `Common Search.alfredworkflow` 文件, 自动完成安装。(使用该插件需要 node 环境)
